1.you need provide a file named xi.ha.exp0.tif     
e.g convent code.jpg xi.ha.exp0.tif

2.run "makebox.sh"+"langstyle" to make box file
e.g  sh makebox.sh chi_sim  

3.use the jTessBoxEditor to correct the wrong words

4.run "creat.sh" to get the final file

5.run "remofile.sh"+"langname"
e.g   sh remofile.sh chi_sim1
